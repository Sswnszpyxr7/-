# 《九命一系:鸣人重生录》

基于 [重生火影.md](重生火影.md) 设计文档实现的命令行文字 RPG。

> 类型:文字冒险 RPG / 角色养成 / 回合制战斗
> 个人自用、非公开发行、非商业传播

## 运行

**统一图形界面**(默认,双击 `启动图形界面.bat` 或):

```bash
cd nine_lives_game
python main.py
```

- `main.py` 是唯一正式入口，GUI 和终端共用 `systems/runtime.py` 的同一套流程
- 基于 Python 自带 tkinter，零运行时第三方依赖
- 左侧剧情窗 + 底部选项按钮(支持数字键 1-9/0、回车继续),右侧边栏实时显示生命/查克拉/经验/命运点/羁绊/契约/道具
- 右下角「⚡ 快速文本」可随时开关逐字打印
- 关闭窗口时自动保存到存档 3,与终端版存档互通

**终端版**:

```bash
cd nine_lives_game
python main.py --cli
```

- 需要 Python 3.10+,无运行时第三方依赖。
- 快速阅读:`python main.py --cli --fast`，或设置环境变量 `NL_FAST=1`。
- 数据校验:`python main.py --validate-data`。

### 系统

- **世界时间**:清晨/白昼/黄昏/夜晚四时段,移动、休息、训练与探索会推进时间,部分见闻只在特定时段出现
- **技能装备栏**:最多装备 6 个已掌握技能,可在系统菜单或每场战斗开始前调整
- **自由组队**:随剧情解锁佐助/小樱/卡卡西/雏田/我爱罗/纲手/小南/带土等同行者,可自由选择最多两人协战
- **地点见闻**:15 个主要地点共 75 个事件；每处包含 2 个日常见闻和一条三阶段连续事件链，受章节/时段/冷却控制，并会反馈队友羁绊、阵营声望、素材与后续文本
- **收藏馆**:GUI 独立页展示角色/敌人/忍术图鉴、成就与结局;收藏进度在三个存档间共享
- **剧情分支图/恋爱阶段图**:收藏馆实时显示当前存档走过的主线、已收集的其他结局、关键角色生死与五条恋爱线的事件/告白进度
- **角色头像与剧情插图**:43 名主要出场角色拥有统一风格头像，GUI 会按最近出现的角色自动切换；15 个地点与波之国/佩恩袭村/忍界大战/真结局等关键段落会显示对应横幅
- **战斗与收藏视觉层**:GUI 支持属性、状态、敌方意图、元素反应、忍具、道具、精英词缀和技能图标；收藏馆包含契约/成就/结局徽章页；16 场主要 Boss 战拥有独立 832×208 切入图
- **地图探索**:木叶 12 地点 + 疾风传 3 地图(砂隐村/妙木山/联军前线),事件随章节动态解锁
- **回合制战斗**:属性克制、状态异常(封印/九尾外衣/恐惧/查克拉紊乱/仙人/九喇嘛模式等)、**队友协战**(佐助/小樱/卡卡西/千代/雏田/我爱罗/带土等随剧情参战)、Boss AI、命运反噬≥3 时敌人强化
- **战术战斗·阶段一**:敌方行动预告、破势槽与行动打断、稳固防御/精准反击、四种主动队友战术，以及生存/破势/捕获等数据化战斗目标
- **战术战斗·阶段二**:森林/浓雾/沙漠/暴雨/妙木山/战争前线环境，湿润→感电、风助火势、蒸汽爆发、泥沼等元素反应，前中后排站位和五项核心忍术二选一强化
- **战术战斗·阶段三**:武器/防具/饰品三槽忍具与九命套装，迅捷/重甲/狂暴/再生/剧毒等精英词缀，五档通缉榜和带挑战修正的新周目继承
- **九命一系**:契约之树、9 级契约等级、**九名契约者**(玖辛奈/琳/纲手/小樱/静音/雏田/小南/照美冥/大地母亲),Lv8【一系】濒死共抗死亡,Lv9【九命同归】
- **契约支援**(每场战斗各一次):赤发守护/温柔医疗/百豪之守/怪力回春/医毒净化/柔拳心眼/纸翼庇护/熔遁掩护/大地之息
- **羁绊联合技**(契约 Lv5 解锁):赤阳封命阵、温柔归命、九命守护、百豪医堂、熔天焦土、桃源归心,以及六线 Lv9 解锁的终式**九命同归**
- **恋爱线**(疾风传开启):小樱/雏田/纲手/小南/琳五条线,各两段约会事件积累「心动值」;大战完结且心动≥50 后可在专属地点告白定情(单恋人制,其余命线温柔收束为家人)。定情后桃源闲话有恋人专属语录,状态页显示恋人
- **忍者成长**:仙人模式/蛙组手/九喇嘛模式/尾兽玉螺旋丸随疾风传剧情习得
- **忍术研究**(§28.1 全部 7 项):多重影分身修炼法(修炼收益×2)、简易封印阵、阳遁恢复术、九尾查克拉外衣、螺旋丸性质变化、螺旋手里剑、仙术基础感知
- **桃源建设**(§28.2 全部 8 项):小屋/医疗室(每日送回复丹)/封印研究间/修炼庭院/温泉/药草田(每日收获)/契约之树深化(升级降价)/命运回廊扩建(免费窥视)
- **命运系统**:命运点经济、反噬 0-6 级(影响战斗难度)、命运回廊随进度更新并可回望已定格结局
- **关系数值**(§21):佐助五维(信任/复仇执念/咒印侵蚀/羁绊/孤立感)与小樱五维(自信/医疗兴趣/信任/执念/责任感),由全程选择塑造,决定终章结局
- **隐藏数值**:暴露度、卡卡西怀疑值、第七班信任、归属感、**九尾波动**(外衣使用累积,可找玖辛奈安抚)
- **任务系统**:主线 8+5(疾风传) + 特殊 4 + 日常 2(找猫/采药)
- **动态委托与阵营声望**:21 个每日讨伐/巡逻/采集模板 + 木叶/砂隐/联军各 5 段阵营故事；声望从陌生晋升至英雄，在 20/50/90/140 点领取补给、专属忍具、属性认可和命运奖励
- **队友成长与联合技强化**:8 名自由队友各有两条成长路线与独立路线奥义，并在羁绊 10/30/60 触发专属事件；联合技按实战次数升至 3 阶，逐阶减耗并强化效果
- **多目标/多阶段 Boss**:波之国双目标、蝎三形态、飞段仪式、角都面具兽、佩恩六道、带土神威和斑三阶段均已实装；生命/查克拉跨阶段保留，契约支援与队友奥义整场共享使用次数
- **忍具锻造与长期经济**:24 个锻造/炼药配方、11 类素材与 33 件忍具；战斗和任务产出「两」，木叶/砂隐/联军商店受声望折扣影响；装备拥有四档品质、+5 强化、8 类词缀，并支持精炼、重铸和拆解
- **契约关键试炼**:九名契约者在晋升 Lv3/Lv5/Lv8/Lv9 前分别完成理解、协同战、抗死与归命试炼；升级不再只是支付命运点
- **蝴蝶效应**:救下的白与再不斩会在木叶崩溃时驰援;感化的我爱罗会在追击战支援;第一部的每个改命,都会在疾风传收到回响——集齐六大改命(长门/带土/阿斯玛/我爱罗/自来也/千代)可达成**九命同归·真结局**
- **存档**:3 槽 + 退出自动存档,旧存档向前兼容

### 美术资源构建

图标与通缉海报的固定提示词清单位于 `output/imagegen/icon-prompts.jsonl`。使用已配置的 GPT Image API 生成原图后，可统一裁切、缩放、合成 Boss 切入图并校验：

```powershell
$env:OPENAI_API_KEY = [Environment]::GetEnvironmentVariable('OPENAI_API_KEY', 'User')
$env:OPENAI_BASE_URL = [Environment]::GetEnvironmentVariable('OPENAI_BASE_URL', 'User')
python "$HOME\.codex\skills\.system\imagegen\scripts\image_gen.py" generate-batch `
  --input output\imagegen\icon-prompts.jsonl `
  --out-dir output\imagegen\icon-sources `
  --concurrency 1 --max-attempts 3 --no-augment
python tools\build_game_art.py --process output\imagegen\icon-sources
```

只重新合成与检查 Boss 切入图：

```bash
python tools/build_game_art.py --compose-boss
```

## 流程指引

1. **第一部**:忍者学校分班 → 对话佐助/小樱 → 训练场铃铛测试 → 回家睡觉(桃源觉醒)
2. **波之国**(Lv4+):火影办公室接任务。想要完美结局:提升与白的羁绊到 2+,攒 4 命运点
3. **琳**:回村后进桃源看「药草田的微光」;医院有老照片、传话、采药、小樱支线
4. **中忍考试**(Lv7+):火影办公室报名。封印术≥10 可在死亡森林彻底挡下咒印;结束直接衔接**木叶崩溃**——救三代需约 3-5 命运点(进言+玖辛奈 Lv3 可降低)
5. **寻找纲手**(Lv10+):木叶大门出发。旅馆选择影响佐助与鼬;回村后多拜访纲手可解锁第三契约
6. **第一部终章**(Lv13+):化解佐助心结(族地/医院事件),然后回家就寝。信任≥55、复仇≤45、咒印=0 可走最优路线
7. **疾风传**(Lv16+ 建议):木叶大门与自来也踏上三年修行 → 砂隐村(风影夺还,攒 3 命运点可救千代) → 火影办公室(晓之阴影) → 妙木山(仙人模式) → 回家就寝(佩恩袭村,备 3+ 命运点) → 木叶大门(忍界大战)
8. **真结局**:六大改命全部达成(我爱罗/千代/阿斯玛/自来也/长门/带土)+ 六名契约者,大战终局解锁「九命同归·真结局」
9. **恋爱线**:疾风传中与小樱(医院/一乐)、雏田(训练场/火影岩)、纲手(办公室/一乐)、小南(木叶大门,佩恩篇后)、琳(桃源)约会;大战完结后回到约会地点告白定情

## 玩法提示

- 桃源修炼有加成;研究「多重影分身修炼法」后收益翻倍
- 联合技在「契约支援」菜单里,契约 Lv5 解锁,每场战斗一次;六线 Lv9 后解锁终式「九命同归」
- 命运反噬≥3 会让敌人变强,可在命运回廊消耗命运点安抚
- 使用九尾外衣会积累九尾波动,≥50 时去桃源找妈妈谈心;与九喇嘛和解后不再积累
- 疾风传的改命节点大多需要 2-3 命运点,进本前记得攒够
- 战斗中先观察敌方意图：高威力攻击适合稳固防御，普通直接攻击可尝试精准反击；使用破势型技能和队友牵制能打断敌方行动
- 前排适合体术与破势，后排适合忍术和安全输出；雨天先用水遁制造湿润，再接雷遁可以触发感电
