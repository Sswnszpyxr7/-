# 火影题材自用文字 RPG 设计文档

# 《九命一系：鸣人重生录》

> 类型：文字冒险 RPG / 角色养成 / 回合制战斗
> 玩法参考：金庸群侠传式自由探索 + 宝可梦式回合战斗
> 使用范围：个人自用、非公开发行、非商业传播
> 主角：漩涡鸣人
> 起点：鸣人重生回到木叶分班的那一天
> 核心设定：鸣人被“地球母亲”赐予契约术【九命一系】，借此改变自己与重要之人的命运。

---

## 1. 游戏核心概念

玩家扮演重生归来的漩涡鸣人，在木叶忍者学校分班当天醒来。

这一世，鸣人不仅保留了前世的记忆，还获得了来自“地球母亲”的契约术——【九命一系】。

这个术并非支配、奴役或强迫他人，而是一种建立在信任、守护、温暖与命运共鸣上的契约。

鸣人可以通过它与未来如同母亲般对待自己、给予自己安全感和温暖的女性建立平等契约，共同改变悲剧命运。

游戏的目标不是单纯变强，而是：

* 挽救遗憾；
* 修复关系；
* 保护重要之人；
* 改写忍界战争的走向；
* 建立属于鸣人与契约者们的精神归处：【桃源】。

---

## 2. 游戏类型定位

### 2.1 核心玩法

本游戏采用文字 RPG 结构，主要由以下部分组成：

1. **地图探索**

   * 类似《金庸群侠传》的区域式自由探索。
   * 玩家可在木叶、训练场、火影岩、死亡森林、波之国、砂隐村等地点移动。
   * 每个地点有 NPC、事件、修炼点、隐藏剧情。

2. **剧情选择**

   * 通过选项推进剧情。
   * 不同选择影响人物好感、命运线、事件结果。

3. **角色养成**

   * 鸣人可提升等级、属性、忍术、体术、查克拉控制、仙术、尾兽力量等。
   * 契约者也可成长，并通过【九命一系】共享部分力量。

4. **回合制战斗**

   * 战斗参考宝可梦的结构。
   * 每回合选择一个行动：忍术、体术、道具、防御、切换支援、逃跑等。
   * 属性克制、状态异常、查克拉消耗、忍具、羁绊技能共同影响战局。

5. **命运改写**

   * 玩家提前知道未来部分事件，可选择干涉。
   * 例如：阻止白死亡、拯救三代火影、改变中忍考试袭击、避免佐助叛逃等。

## 3. 世界观改造

### 3.1 重生起点

鸣人在第四次忍界大战结束多年后，因为某种未知原因灵魂回到过去。

醒来时间点：

> 木叶忍者学校分班当天，伊鲁卡即将宣布第七班成员。

鸣人拥有：

* 前世记忆；
* 战斗经验残片；
* 对未来大事件的认知；
* 被封印削弱后的九尾查克拉感知；
* 新能力：【九命一系】。

---

## 4. 核心设定：【九命一系】

## 【九命一系】

### 4.1 契约来源

【九命一系】是地球母亲赐予鸣人的特殊契约术。

它并不是传统忍术，也不是血继限界，而是一种与生命、命运、亲情、守护意志相关的特殊力量。

这个术的本质是：

> 将原本孤独的鸣人，与真正愿意给予他温暖、安全感和母性守护的人连接在一起，使他们共同拥有改变命运的力量。

---

### 4.2 契约对象

契约对象上限：**9人**

其中固定存在：

1. 漩涡玖辛奈
2. 野原琳

其余契约对象需满足以下条件：

* 必须是女性；
* 必须在未来或当前世界线中，如同母亲、姐姐、守护者一般对待鸣人；
* 与鸣人的关系必须平等、温馨、信任；
* 关系基础必须是安全感、守护、陪伴，而不是控制或利用；
* 契约不能强迫缔结；
* 契约者必须在心灵层面认可鸣人；
* 鸣人也必须真心愿意守护对方。

### 4.3 禁止条件

以下情况无法契约：

* 对方被幻术、洗脑、操控；
* 对方出于恐惧、利益、威胁而同意；
* 鸣人试图用契约支配他人；
* 双方缺乏真实信任；
* 对方对鸣人没有亲情式守护意愿。

---

## 5. 契约效果

### 5.1 女性侧效果

#### 生命与感知

契约女性会与鸣人以及其他契约女性建立生命共鸣。

效果包括：

* 生命力提升；
* 伤势恢复速度提升；
* 查克拉总量缓慢增加；
* 忍术天赋轻微提升；
* 感知鸣人生命状态；
* 在危急时感知鸣人的情绪波动；
* 可在濒死状态下被【九命一系】拉回一次，但需要消耗大量共鸣能量。

#### 力量与特异性

契约者之间可有限度借用彼此力量与天赋。

例子：

* 鸣人可借用玖辛奈的封印术理解力；
* 玖辛奈可借用鸣人的阳遁生命力；
* 琳可借用鸣人的感知能力；
* 契约者之间可共享部分抗性；
* 契约者可以获得专属支援技能。

---

### 5.2 鸣人侧效果

鸣人通过契约获得：

* 生命力上限提升；
* 查克拉恢复提升；
* 情绪稳定性提升；
* 封印术学习速度提升；
* 感知恶意能力提前觉醒；
* 与契约者之间的羁绊技能；
* 可进入专属小世界【桃源】；
* 可在关键剧情中触发“命运改写判定”。

---

### 5.3 契约等级

每名契约者拥有独立契约等级。

| 契约等级 | 名称   | 解锁效果         |
| ---- | ---- | ------------ |
| 1    | 初契   | 可感知彼此生命状态    |
| 2    | 同心   | 可共享少量生命力     |
| 3    | 牵命   | 可借用一项基础天赋    |
| 4    | 共鸣   | 解锁支援技能       |
| 5    | 命桥   | 可在战斗中发动联合技能  |
| 6    | 归心   | 可进入【桃源】长时间休养 |
| 7    | 命织   | 可干涉小型命运事件    |
| 8    | 一系   | 可共同抵御死亡命运    |
| 9    | 九命同归 | 解锁终极命运改写技能   |

## 6. 专属小世界：【桃源】

## 【桃源】

【桃源】是鸣人与九位契约女性的专属温馨空间。

这里不是战斗基地，也不是权力象征，而是一个让所有契约者都能安心休息、交流、疗伤和生活的小世界。

### 6.1 进入方式

* 鸣人可主动进入；
* 契约者可随时随地传送进入；
* 战斗中无法随意进入，除非触发特殊剧情；
* 濒死状态下可能被自动转移进入【桃源】急救。

---

### 6.2 外部环境

【桃源】拥有不断变化的美丽景观：

* 花海；
* 溪流；
* 雪山；
* 沙滩；
* 竹林；
* 星空湖；
* 温泉；
* 枫叶小径；
* 月光草原。

环境会根据契约者心情与剧情进度变化。

例如：

* 玖辛奈苏醒后，桃源会出现红枫林；
* 琳入住后，桃源会出现药草田；
* 鸣人情绪低落时，天空会下细雨；
* 羁绊提升后，夜空会出现新的星座。

---

### 6.3 内部设施

核心建筑是一座温馨紧凑、设备齐全的小屋。

小屋设施包括：

* 客厅；
* 厨房；
* 茶室；
* 书房；
* 医疗室；
* 封印术研究间；
* 修炼庭院；
* 温泉浴池；
* 储物间；
* 契约之树；
* 命运回廊。

---

### 6.4 功能系统

#### 休息恢复

进入【桃源】后可以恢复：

* 生命值；
* 查克拉；
* 精神状态；
* 负面状态；
* 部分剧情创伤。

#### 修炼加成

在【桃源】修炼可获得额外收益：

* 封印术经验提升；
* 医疗忍术经验提升；
* 查克拉控制提升；
* 羁绊经验提升；
* 契约技能熟练度提升。

#### 契约之树

契约之树是【九命一系】的核心显化。

功能：

* 查看契约者状态；
* 提升契约等级；
* 解锁羁绊技能；
* 查看命运节点；
* 消耗命运点改写事件。

#### 命运回廊

命运回廊显示未来可能发生的重大事件。

例如：

* 波之国任务；
* 中忍考试；
* 木叶崩溃计划；
* 佐助离村；
* 晓组织行动；
* 佩恩袭村；
* 第四次忍界大战。

玩家可提前查看模糊提示，并通过行动改变结果。

## 7. 主角设定：漩涡鸣人

### 7.1 初始状态

时间点：分班当天。

鸣人表面仍是忍校毕业生，但实际拥有前世经验。

不过为了避免过强破坏平衡，设定为：

* 记忆完整；
* 身体仍是少年身体；
* 查克拉控制较差；
* 九尾力量仍受封印影响；
* 前世高阶技能需要重新修炼；
* 仙术、尾兽模式、六道力量暂时无法使用；
* 情绪成熟度提升，但偶尔仍会受少年身体影响。

---

## 8. 战斗系统设计

### 8.1 战斗基础

战斗采用回合制。

每回合玩家可选择：

1. 普通攻击；
2. 忍术；
3. 体术；
4. 幻术；
5. 封印术；
6. 忍具；
7. 防御；
8. 切换支援；
9. 契约技能；
10. 逃跑。

---

### 8.2 属性类型

参考宝可梦，忍术与角色拥有属性类型。

| 属性 | 克制       | 被克制     |
| -- | -------- | ------- |
| 火  | 风、虫、木    | 水、土     |
| 风  | 雷、毒、音    | 火       |
| 雷  | 水、土      | 风       |
| 水  | 火、土      | 雷       |
| 土  | 雷、火      | 水       |
| 木  | 水、土      | 火、雷     |
| 阴  | 精神、幻术    | 阳       |
| 阳  | 生命、恢复    | 阴       |
| 封印 | 尾兽、秽土、咒印 | 高速体术    |
| 医疗 | 伤势、毒     | 爆发伤害    |
| 体术 | 幻术型、脆皮   | 封印、反击   |
| 幻术 | 低精神目标    | 高精神、写轮眼 |
| 尾兽 | 大范围敌人    | 封印术     |

---

### 8.3 状态异常

| 状态    | 效果            |
| ----- | ------------- |
| 灼烧    | 每回合损失生命，攻击下降  |
| 麻痹    | 速度下降，概率无法行动   |
| 中毒    | 每回合损失生命       |
| 混乱    | 概率攻击自己或队友     |
| 恐惧    | 无法使用高级技能      |
| 封印    | 无法使用部分忍术      |
| 流血    | 受到体术伤害增加      |
| 查克拉紊乱 | 忍术消耗增加        |
| 精神动摇  | 幻术抗性下降        |
| 命运锁定  | 关键剧情中无法逃避命运判定 |

---

### 8.4 战斗公式

基础伤害公式：

```text
伤害 = 技能威力 × 攻击属性 / 防御属性 × 属性克制 × 随机浮动 × 状态修正
```

推荐简化为：

```text
最终伤害 = 技能威力 × 攻击方主属性 ÷ 防御方对应属性 × 克制倍率
```

克制倍率：

| 情况  |  倍率 |
| --- | --: |
| 克制  | 1.5 |
| 强克制 | 2.0 |
| 普通  | 1.0 |
| 被抵抗 | 0.5 |
| 无效  |   0 |

## 9. 队伍与支援系统

### 9.1 出战结构

玩家主控鸣人。

战斗结构可设计为：

* 鸣人主战；
* 最多 2 名队友协战；
* 最多 3 名契约者作为支援；
* 特定剧情中契约者可临时参战。

### 9.2 支援技能

契约者可以提供支援技能。

例子：

| 契约者      | 支援技能  | 效果               |
| -------- | ----- | ---------------- |
| 玖辛奈      | 赤发守护  | 为鸣人提供护盾，并提升封印术威力 |
| 琳        | 温柔医疗  | 回复生命，解除中毒与流血     |
| 自定义契约者 A | 查克拉安抚 | 降低九尾暴走风险         |
| 自定义契约者 B | 精神共鸣  | 提高幻术抗性           |
| 自定义契约者 C | 命运牵引  | 关键回合避免致命伤        |

## 10. 角色关系系统

### 10.1 好感与信任

每个重要角色有两个核心数值：

| 数值   | 说明           |
| ---- | ------------ |
| 好感   | 对鸣人的亲近程度     |
| 信任   | 是否愿意相信鸣人的判断  |
| 安全感  | 对鸣人是否感到安心    |
| 命运共鸣 | 是否可触发契约或命运改写 |

契约前置条件建议：

```text
好感 ≥ 70
信任 ≥ 80
安全感 ≥ 80
命运共鸣 ≥ 60
完成专属剧情
```

### 10.2 关系事件

关系可通过事件提升：

* 共同修炼；
* 送礼；
* 对话；
* 关键剧情救援；
* 战斗中保护对方；
* 桃源中交流；
* 解开对方心结；
* 改变对方原本悲剧命运。

## 11. 剧情主线结构

### 第一章：重生分班

关键事件：

* 鸣人醒来；
* 发现自己回到分班日；
* 伊鲁卡宣布第七班；
* 鸣人重新见到佐助、小樱、卡卡西；
* 【九命一系】第一次觉醒；
* 桃源雏形出现；
* 鸣人意识到玖辛奈的残魂仍在封印深处。

目标：

* 通过卡卡西铃铛测试；
* 不暴露全部实力；
* 尝试改变第七班关系；
* 解锁九命系统。

---

### 第二章：第七班重塑

关键事件：

* 铃铛测试；
* 鸣人主动团结佐助与小樱；
* 卡卡西察觉鸣人的异常成熟；
* 鸣人尝试提前改善佐助心态；
* 初次进入桃源；
* 与玖辛奈残魂短暂对话。

目标：

* 第七班信任度达到 50；
* 鸣人查克拉控制提升；
* 解锁基础命运点。

---

### 第三章：波之国任务

关键事件：

* 达兹纳护送任务；
* 再不斩袭击；
* 白登场；
* 鸣人尝试避免白与再不斩死亡；
* 命运第一次强烈反噬。

目标：

* 击败再不斩；
* 说服白；
* 改变白死亡命运；
* 解锁“命运反噬”系统。

可能结局：

1. 原剧情结局：白与再不斩死亡；
2. 半改写结局：白存活，再不斩死亡；
3. 完美改写结局：白与再不斩存活，并离开雾隐追杀体系。

---

### 第四章：中忍考试

关键事件：

* 笔试；
* 死亡森林；
* 大蛇丸袭击；
* 佐助咒印；
* 我爱罗登场；
* 木叶崩溃计划前兆。

目标：

* 降低佐助黑化值；
* 避免小樱重伤；
* 提前布置木叶防御；

## 12. 命运系统

### 12.1 命运点

命运点是游戏核心资源。

获得方式：

* 完成主线任务；
* 改变悲剧事件；
* 提升契约等级；
* 战胜强敌；
* 解开心结；
* 拯救关键人物；
* 在不暴露重生秘密的情况下完成正确行动。

消耗方式：

* 预知未来片段；
* 降低命运反噬；
* 触发特殊选项；
* 在剧情死亡节点进行改写；
* 强化【九命一系】；
* 扩展【桃源】功能。

---

### 12.2 命运反噬

改变重大剧情会引发反噬。

反噬表现：

* 敌人提前行动；
* 原本不会出现的人物提前登场；
* 鸣人身体负担增加；
* 九尾封印波动；
* 某些角色怀疑鸣人；
* 新的危险替代旧的悲剧。

反噬等级：

| 等级 | 表现         |
| -- | ---------- |
| 1  | 小事件偏移      |
| 2  | 敌人警觉       |
| 3  | 战斗难度提高     |
| 4  | 剧情提前爆发     |
| 5  | 关键人物死亡风险增加 |
| 6  | 世界线强制修正    |

    while player.hp > 0 and enemy.hp > 0:
        show_battle_status(player, enemy)
        action = choose_action(player)

        player_result = execute_action(player, enemy, action)

        if enemy.hp <= 0:
            return "win"

        enemy_action = enemy_choose_action(enemy, player)

## 29. 设计原则

开发过程中建议遵守以下原则：

1. **先做能玩的，再做复杂的。**
2. **剧情分支不要一开始写太多。**
3. **每章只设置 1 到 2 个重大改命点。**
4. **战斗数值宁可简单，也要稳定。**
5. **契约系统必须强调平等、信任、温暖。**
6. **桃源不是后宫空间，而是家、疗愈所和精神归处。**
7. **鸣人不能开局无敌，否则剧情会失去张力。**
8. **改变命运必须付出代价，否则系统没有重量。**

## 30. 总结

《九命一系：鸣人重生录》是一款以鸣人重生为核心的文字 RPG。

它的玩法可以分成四条主线：

```text
探索木叶与忍界
培养鸣人与同伴
通过战斗与选择改变剧情
通过九命一系建立温暖的命运共同体
```

最推荐的开发方式是：

```text
先用 Python 做命令行文字版。
先完成分班日、铃铛测试、桃源觉醒。
再逐步扩展波之国、中忍考试、木叶崩溃等篇章。
```

游戏的核心体验应当是：

> 鸣人带着记忆重新开始，不再孤独地追逐认可，而是主动守护、修复遗憾，并与真正给予他温暖的人一起改写命运。
